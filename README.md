# TCP Port Scanner (Python)

Um scanner de portas TCP desenvolvido em Python para análise de redes e aprendizado em cibersegurança.

---

## Sobre

Este projeto implementa um scanner básico de portas TCP que tenta estabelecer conexões com um host alvo em uma faixa de portas, com o objetivo de identificar quais estão abertas.

É voltado para prática e compreensão de:
- Comunicação TCP/IP
- Programação com sockets em Python
- Técnicas básicas de reconhecimento de rede

---

## Funcionalidades

- Varredura de portas em um intervalo (padrão: 1–65535)
- Identificação de portas TCP abertas
- Leve e simples de executar
- Utiliza a biblioteca nativa `socket` do Python
